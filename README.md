# JAVASCRIPT_SMIT_BATCH_18_ASSIGNMENT_7_STRING_METHODS_20_TO_25

## Description
This repository contains 24 JavaScript exercises on string methods (Chapters 20–25) for SMIT Batch 18.  
Each exercise is beginner-friendly, using `var`, `prompt`, and `alert`.  
All exercises include external `script.js` and formatted `index.html`.  
Topics covered: string concatenation, length, index, replace, search, conversion, and validation.
